# Homebrew Tap for ManagedSSH

Install ManagedSSH with Homebrew:

```bash
brew tap mylovelytools/managedssh
brew install mylovelytools/managedssh/managedssh
```

The formula builds from the stable release source tarball and installs the `managedssh` binary, the `managedssh.1` man page, and the `LICENSE` file.
